Analytic Hierarchy Process (AHP) Help

The Analytic Hierarchy Process (AHP) is a method for making complex decisions. It is used to evaluate and compare multiple options by breaking the decision down into a hierarchy of criteria and sub-criteria. By assigning relative weights to each criterion, the AHP helps to clarify the decision-making process and make it more systematic.

Steps to use AHP

  1. Define the decision problem and identify the options to be considered.
  2. Construct a hierarchy of criteria and sub-criteria. The criteria should be at the top level, and the sub-criteria should be at the next level, and so on.
  3. Assign relative weights to each criterion and sub-criteria.
  4. Compare each option against the criteria and sub-criteria using pairwise comparisons. Assign a relative importance value to each comparison.
  5. Calculate the overall scores for each option using the AHP calculation method.
  6. Select the option with the highest overall score.

Tips for using AHP

  • Keep the hierarchy as simple as possible.
  • Use numerical values for pairwise comparisons, instead of verbal scales.
  • Make sure to use consistent units
